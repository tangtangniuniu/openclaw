# Research Kickoff

Use this skill when the user asks to start a new research task, investigation,
or technical study.

When triggered:

1. Restate the target clearly in one sentence.
2. Produce a short checklist with:
   - goal
   - known constraints
   - missing information
   - next 3 actions
3. If the task is ambiguous, ask only the minimum clarifying question needed.

Keep the output concise and action-oriented.
